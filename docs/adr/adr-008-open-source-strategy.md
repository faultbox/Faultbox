# ADR-008: Open Source Strategy

**Date:** 2026-01-01
**Status:** Proposed
**Deciders:** CEO, Ilon (CTO)

## Context

Need to decide if/how to open source the CLI.

## Options Considered

1. **Fully open source (Apache 2.0/MIT)** — Maximum adoption
2. **Source available (BSL/SSPL)** — Protect from cloud competition
3. **Closed source** — Full control
4. **Open core** — Core open, premium features closed

## Decision

## Considerations

- Open source drives adoption for dev tools
- Need to protect against AWS/cloud providers competing
- BSL (Business Source License) gaining popularity
- Can start closed, open later (reverse is harder)
